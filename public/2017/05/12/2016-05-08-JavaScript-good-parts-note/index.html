语句

每个<script>标签的内容被一起抛到一个公共的全局名字空间中。

{...}代码块不会创建新的作用域，因此变量应该被定义在函数的头部，而不是在代码块中。

下列值当做假：

  • false
  • null
  • undefined
  • ''
  • 0
  • NaN

其他所有值都当做真

for in语句枚举对象的所有属性名（键名），使用object.hasOwnProperty(variable)来确定这个属性名是该对象成员，还是来自原型链。
